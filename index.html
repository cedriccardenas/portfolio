Project

Log File Parser

Developed in Python, this command-line tool was designed to scrutinize log files for HTTP requests from clients to the server. During my tenure at Change Healthcare, the absence of existing tools for parsing client-side logs prompted the creation of this solution.

The tool employs a logic that interprets regular expressions, extracting information and storing it in a list of dictionaries. It systematically searches through all client-side logs, identifying matching search strings, and organizes the output into a structured table for comprehensive analysis.
